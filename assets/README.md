# Local build assets

This directory is for files that are required to build firmware but cannot be committed.

Required for `firmware` and `all` modes:

- NTR blowfish source: either `biosnds7.rom` or `ntrBlowfish.bin`
- TWL blowfish source: either `biosdsi7.rom` or `twlBlowfish.bin`

Optional for `--wrfuxxed` builds:

- `wrfu.srl` (WRFU Tester v0.60)

Do not commit copyrighted assets.
